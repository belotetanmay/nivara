# NIVARA Release Candidate (v1.0.0-RC) Release Notes

**Build Version**: `1.0.0`
**Target Branch**: `phase6-production-services`
**Release Date**: July 2026

---

## 🌟 Highlights

- **Complete Customer & Partner Workflows**: Search, book, customize, and complete mobile recovery sessions seamlessly.
- **Social Sign-In**: Integrated Google OAuth and Apple Sign-In alongside standard email/password and OTP reset.
- **Real Razorpay Gateway**: HMAC-SHA256 signature verification supporting UPI, Credit Cards, Debit Cards, Net Banking, and Wallets.
- **Master Legal Policy Implementation**: Full DPDP Act 2023 compliant Privacy Policy, Terms & Conditions, Cancellation Policy, and Grievance Redressal.
- **Smooth Keyboard Experience**: Unified `FormContainer` keyboard avoiding view prevents input overlap.
- **Offline Reliability**: Real-time network banner with instant connection retry handler.
- **80/20 Partner Financial Model**: Automated 80% host payout calculations and platform commission tracking.
